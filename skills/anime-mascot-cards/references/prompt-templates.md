# 提示词模板库

本文件提供三类模板:**① 角色定稿提示词**(先生成参考图)、**② 模板块**(整套卡固定的版式骨架)、**③ 成卡提示词**(每张卡 = 角色块 + 模板块 + 内容字段 + 约束句)。文末是几种风格变体(手绘卡 / chibi / 设定表 / 贴纸网格)。

约定:
- `【固定·X】` = 整套卡逐字不变的部分(角色 / 风格 / 版式)。
- `{{ }}` = 需要替换的字段;`{{角色…}}` 在角色定稿后也变成固定值。
- 主打 gpt-image,所有模板已按 gpt-image 习惯写(竖版 `1024x1536`、自然语言一致性约束、不带 MJ 参数)。换 Nano Banana 时把尺寸句改成 "9:16",其余照用。

---

## ① 角色定稿提示词(生成参考图,只用一次)

目的:先单独生成一张高质量看板娘立绘,作为后续每张卡的"锚点参考图"。

```text
为一套知识科普卡片设计一位二次元看板娘角色,请生成她的高质量半身 / 全身立绘:

角色设定:{{一句话人设,贴合主题,例如:一位热爱编程的科技感少女}}
外观:
- 发型 / 发色:{{如:银色双马尾,发梢微卷}}
- 瞳色:{{如:湖蓝色,大而有神}}
- 服装:{{如:白色连帽卫衣 + 蓝色细节,领口系数据线造型挂饰}}
- 配饰:{{如:半透明 AR 眼镜 / 发夹}}
- 气质表情:{{如:亲和、自信微笑,面向读者}}

画风:{{统一画风词,如:高质量 2D 日系插画,线条干净利落,赛璐珞上色,色彩温暖明快}}
配色基调:{{主色 + 辅色,给十六进制,如:奶油白 #F3F0E9 / 科技蓝 #3B82F6}}
画面:纯色或简洁背景,角色清晰居中,便于作为后续卡片的角色参考图。
尺寸:竖版 1024x1536。高清,干净专业。
```

> 可选:再生成一张"角色设定表 / 三视图"(模板见文末变体 C),把角色外观钉得更死,后续更不易漂。

## ② 模板块(整套卡固定的版式骨架)

把它写进角色档案,之后每张成卡提示词都逐字包含它。

```text
【固定·版式】创作一张二次元手绘风格的知识科普卡片,竖版 1024x1536。
- 背景:{{带轻微肌理的底色,如:米白色 #F3F0E9,质朴亲切}}。
- 顶部:醒目的大号中文标题区,{{标题样式,如:红黑对比的手写体}},作为视觉焦点。
- 正文:分为 2 至 4 个清晰小节,每节用简短精炼的中文短语表达一个要点;小节之间留白,配可爱的手绘小图标 / 箭头引导线。
- 看板娘:半身像固定置于卡片{{位置,如:右下角}},约占画面宽度 1/3,与文字区视觉平衡。
- 画风:{{统一画风词,与角色定稿一致}}。
- 配色:整体遵循 {{主色 / 辅色十六进制}}。
【固定·收尾词】中文标注、文字清晰可读、画面干净专业、留白充足、高清。
```

## ③ 成卡提示词(每张卡的最终提示词)

这是真正粘贴到 gpt-image 的东西。装配方式 = 角色块 + 模板块 + 内容字段 + 约束句。

```text
【固定·角色】画面中固定出现同一位看板娘:{{角色定稿后的完整外观描述,逐字复制角色档案}}
(务必与上传的参考图 100% 一致:面部特征、发型、发色、瞳色、服装、配饰、配色完全相同)。

{{此处逐字粘贴"② 模板块"全文}}

【本张·知识内容】
标题:{{这张卡的主题,例如:什么是扩散模型?}}
副标题:{{可选,一句点睛 / 比喻,例如:像往清水里滴墨又倒放}}
要点:
1) {{要点 1,一句话}}
2) {{要点 2,一句话}}
3) {{要点 3,一句话(2–4 条,按内容增减)}}

【固定·一致性约束】请保持与参考图中角色完全一致——面部、发型、发色、服装、配色、画风 100% 相同,仅更换上方文字内容与角色的自然姿态 / 表情。{{第 2 张起追加:并保持风格、版式、配色与上一张卡片完全一致。}}
```

**批量做法**:把"角色块 / 模板块 / 约束句"当成不可改的常量,新建一张卡时只重写"【本张·知识内容】"三到四行。这样每张卡的差异被压到最小,统一性最高。

## 风格变体

主流程默认用"手绘信息图"风(变体 A)。按主题可换:

### 变体 A · 手绘信息图卡(默认 · 最通用)

即上面 ② 的写法。特征:米白纸感背景、手写标题、手绘小图标、分节要点。适合绝大多数知识科普。

### 变体 B · chibi Q版卡(轻松 / 儿童 / 趣味科普)

把模板块的画风改为:

```text
画风:chibi Q版,夸张有神的大眼睛、柔和圆润的线条、头身比 ~2.5,糖果色系明快配色。
看板娘为 Q版全身,动作活泼(如指着要点 / 比手势)。
```

### 变体 C · 角色设定表 / 三视图(用于"定角色"或做"角色介绍卡")

```text
你是顶尖的动漫概念美术设计师,擅长制作详尽的角色设定图(Character Sheet)。
请生成一张竖版 1024x1536 的"看板娘角色设定卡":
- 中心:角色全身立绘作为视觉锚点。
- 四周环绕展示:服装分层(外套 / 内搭 / 鞋)、3-4 个表情(开心 / 害羞 / 惊讶 / 思考)、关键配饰材质特写 1-2 个。
- 用手绘箭头 / 引线连接到角色对应部位,旁边模拟手写中文注释,简述材质 / 名称。
- 背景:米黄 / 羊皮纸纹理,设计手稿氛围;画风:高质量 2D 插画,线条干净利落。
角色主体 = {{角色描述或上传参考图}}。中文标注,高清,严格竖版 1024x1536。
```

### 变体 D · 表情包 / 贴纸网格(一图成套)

```text
以参考图中的看板娘为主角,创作一套 chibi 表情贴纸,共六格(2x3 网格):
1) 比剪刀手俏皮眨眼;2) 泪眼汪汪;3) 张臂拥抱;4) 托腮甜笑;5) 自信指向前方带闪光;6) 飞吻飘爱心。
统一美学:保留 chibi 风格、{{角色发型发色}}、{{角色服装}};背景统一 {{底色}},点缀星星 / 彩纸屑,每格周围留白。
保持人物与参考图特征完全一致。竖版 1024x1536。
```

> 贴纸网格靠"单图内由模型自我约束"实现一致,适合一次出一组表情;成系列知识卡仍走 ①②③ 的"以图带图"路线。

## 装配自查清单

生成成卡提示词后,逐条核对:

- [ ] 角色块在每张卡里**逐字相同**(没顺手改字)?
- [ ] 模板块在每张卡里**逐字相同**?
  > 角色块 / 模板块往往很长,**人眼极易漏掉一个标点 / 空格的差异**。把各卡这两块抽出来用 `diff` 或脚本逐字节比对,比肉眼可靠得多——这是验证"统一"是否真的成立的最硬手段。
- [ ] 只有"【本张·知识内容】"每张不同?
- [ ] 有一致性约束句?第 2 张起有"与上一张一致"?
- [ ] 尺寸是 gpt-image 支持的档位(竖版 `1024x1536`),没误写 9:16 / `--ar`?
- [ ] 没混入 MJ 参数(`--seed` / `--cref` / `--iw`)?
- [ ] 配色给的是十六进制而非"暖色调"这类模糊词?
